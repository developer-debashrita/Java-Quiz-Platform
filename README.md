# ☕ Java Mastery Quiz Platform

An interactive and responsive Java Quiz Platform built using **HTML, CSS, and JavaScript**. This project helps students and beginners strengthen their Java programming knowledge through multiple-choice questions covering Java fundamentals, Object-Oriented Programming (OOP), Spring Boot, REST APIs, debugging, and database concepts.

---

## 📌 Features

- 📚 81+ Java MCQs with explanations
- 🎯 Topic-wise quiz selection
- 🔀 Shuffle questions randomly
- 📊 Live score and accuracy tracking
- 📈 Progress bar
- ✅ Instant answer validation
- 💡 Explanation for every answer
- 📱 Fully responsive UI
- 🌙 Modern glassmorphism interface
- 🚀 Career roadmap section for Java developers

---

No installation or server is required.

---

## 🎮 How to Use

1. Select a topic or choose **All Topics**.
2. Click **Start / Restart Quiz**.
3. Answer each multiple-choice question.
4. View instant feedback and explanations.
5. Track your score and accuracy.
6. Complete the quiz to view your final result.
